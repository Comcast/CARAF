# CARAF Changes

This file gives an overview of the changes in each version of CARAF.

## PQBench Releases
- [CARAF 1.0](#caraf-10)
- [CARAF 2.0](#caraf-20)

- [CARAF 3.2](#caraf-32)

## CARAF 1.0
This release includes the 5 phases of CARAF with knowledge base that includes articles related to PQC migration.

## CARAF 2.0
This CARAF 2.0 release includes:

1. Simplification of CARAF phases from 5 phases into 3 phases to allow for clearer definitions of each phase, input, output, and actionable items.
2. CARAF calculator that demonstrates the risk assessment process.

## CARAF 3.0
This version includes the addition of [PQBench](./Tools/PQBench/CHANGES.md) into CARAF.

## CARAF 3.1
This version includes the addition of [PQBench](./Tools/PQBench/CHANGES.md) into CARAF.

## CARAF 3.2
This version simplifies CARAF Phase 1 questions: 

1. All questions are now under one category.
2. All questions have consistent scoring: "Yes" means +1.
